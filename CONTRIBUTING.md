# Contributing to QzRAM 🚀

Thank you for your interest in QzRAM! This is a 100% open-source project, and we warmly welcome contributions from the Linux kernel and systems programming community.

## How to Contribute

1. **Fork the Repository:** Create your own copy of this repository on GitHub.
2. **Clone and Test:** Download your fork locally, make your changes, and compile using the flat-structured `Makefile`.
3. **Submit a Pull Request (PR):** Send your clean, well-documented changes back to our main branch.

## Code of Conduct & Guidelines
- Ensure all source code remains 100% kernel-safe, pure integer arithmetic with zero floating-point (FPU) operations.
- Avoid introducing nested directories or complex folder structures; we prefer to keep the repository flat and highly scannable.
- Respect the GPL-2.0 license constraints.
